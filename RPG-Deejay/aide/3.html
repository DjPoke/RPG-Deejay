Raccourcis clavier

Voici la liste des raccourcis clavier de RPG Deejay :

Echap : Pour stopper tous les sons joués.
Suppr : Supprimer le son sélectionné dans la liste qui a le focus.
Entré : joue le son sélectionné.
Fin : éteint la piste d'ambiance en fondu. (En travaux !)
Control+N : Nouvelle playlist.
Control+O : Ouvrir une playlist.
Control+S : Enregistre la playlist à l'endroit de votre choix.
Control+I : Importer un fichier audio.
Control+D : Importer un dossier de fichiers audios.
Control+U : Ajouter une URL à jouer sans la piste de musiques internet.
Control+Q : Quitter le logiciel.
F1 : Aide de ce logiciel.
